                <h3>รายละเอียดเกี่ยวกับโปรแกรม</h3>
                <p>
                    Taskmanager คือเครื่องมือที่ใช้สำหรับจัดการข้อมูลงานต่างๆ(Task) ผู้ใช้จะสามารถบันทึกข้อมูลงาน (Add Task) ด้วยการใส่ชื่อและรายละเอียดของงาน
                    โดยงานที่ถูกเพิ่มเข้าไป จะมีสถานะ "ยังไม่เสร็จงาน"(Completed : False) เมื่อผู้ใช้ได้ทำงานเสร็จเรียบร้อยแล้ว ผู้ใช้งานสามารถใช้ฟังก์ชัน "Commit" เพื่อเปลี่ยนสถานะเป็น "เสร็จแล้ว"(Completed : True)
                    นอกเหนือจากนั้นผู้ใช้สามารถที่จะใช้งานฟังก์ชัน อื่นๆ ได้แก่ "Delete Task" ลบข้อมูลงานนั้นๆออก , "Search Task" ค้นหารายละเอียดของงานด้วยชื่อ ,"Show all Task" แสดงรายละเอียดของทุกงาน ที่ถูกบันทึก
                    และ "Edit Task" สามารถแก้ไขชื่อ,รายละเอียด และ รีเซ็ต สถานะของงาน จาก "เสร็จแล้ว" เป็น "ยังไม่เสร็จงาน" ได้
                </p>
